## 🚀 Getting Started

### 1. 依存関係をインストール
```
npm install
```

### 2. 開発サーバーの起動
```
npm run dev
```

### 3. Storybookの起動
```
npm run storybook
```

### ビルド
```
npm run build 
```
ビルド結果は dist/ に出力されます。

```
npm run build-storybook
```
静的な Storybook は storybook-static/ に出力され、Netlify/Vercel などでもホスト可能です。
